# C++ MySQL数据库连接池

## 项目简介
一个基于 **C++11** 实现的高性能MySQL数据库连接池，用于解决频繁创建数据库连接的性能问题。通过连接复用、动态扩容和空闲连接回收等机制，相比直接连接数据库**性能提升约40%**。

## 核心特性
-  **连接复用**：避免频繁创建/销毁连接
-  **动态扩容**：按需创建新连接，最多支持1024个连接
-  **智能回收**：自动回收空闲超时连接
-  **线程安全**：使用互斥锁+条件变量+智能指针
-  **配置灵活**：通过配置文件管理连接参数
-  **单例模式**：全局唯一连接池实例



### 环境要求
- C++11 或更高版本
- MySQL Server
- MySQL Connector/C 开发库
